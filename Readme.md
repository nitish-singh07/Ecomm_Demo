## Quick Start

1. **Clone the repository:**

   ```bash
   $ git clone https://github.com/nitish-singh07/Ecomm_Demo.git
   ```

2. **Go into the repository:**

   ```bash
   cd Ecomm_Demo
   ```

3. **Install dependencies**

   ```bash
   $ npm install
   ```

4. **_For running the app:_**
   ```bash
   $ npx expo start
   ```
